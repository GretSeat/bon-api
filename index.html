Bon Appetit Menu API

This is an API that has all of the Bon Appetit Menu Stuff

/api gets you into the entire menu

/api/drinkMenu gets you into the drink menu items

/api/foodMenu gets you into just the food menu items

/api/foodMenu/:courseType is where you would go to get specific information about one course. Replace :courseType with "specials", "appetizers", "entrees", "salads", "lighterSelections", "sides" or "desserts"
